The Player ID

SELECT id FROM players WHERE LOWER(username) = '$playerUsername'

The order of operations for this web-query is as follows:

Query the selection from the `players` database table.

If the entered `playerUsername` field matches `username` in the `players` table, return the player's `id`.

If the entered `playerUsername` field does not match any `username` in the `players` table, send an error.

Looking up a player in the `players` table of the database is a very straightforward process. Enter a player's username to find their ID. The username field is insensitive to case. The query only works using the following information:

  • Username (str/text)
  • id (int)

The lookup field will work with numbers, letters, underscores and spaces.
