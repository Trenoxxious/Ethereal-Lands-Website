The Ethereal Soul

UPDATE etherealsouls SET amount = `$newSouls` WHERE id = `$playerID`

The order of operations for this web-query is as follows:

Query the `etherealsouls` database table for a row matching `playerID`.

If a row is found, we update the `amount` in the table to be `$newSouls`.

If a row is not found, send an error.

Adjusting Ethereal Souls for the player is a very simple query. We will take the amount entered by the player, find the amount of souls the player has in the database and alter it using the entered data. There are two columns used for this query in the `etherealsouls` database table:

  • id (int)
  • amount (int)

A positive integer (eg. 100) will add Ethereal Souls to the player's current amount. A negative integer (eg. -100) will subtract Ethereal Souls from the player's current amount. The amount of Ethereal Souls will not go below 0 under any circumstances.
